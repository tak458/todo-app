# todo-app

ブラウザのみで完結する Todo アプリ

GitHub IO : https://tak458.github.io/todo-app/

## Project setup

```shell
npm ci
```

### Compiles and hot-reloads for development

```shell
npm run dev
```

### Compiles and minifies for production

```shell
npm run build
```

### Build Storybook

```shell
npm run build-storybook
```

### Run your tests

```shell
npm run test
```

### Lints and fixes files

```shell
npm run lint
```
